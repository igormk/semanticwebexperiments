The research is in the form of series of experiments which are used
to test the Jena Framework. From every experiments there will be conclusions.
Every experiment will be placed in different folder with Experiment[number] as
the name that defines it.
This project is for noncommercial purposes and is used for getting to know
how the semantic web software actually works and testing its performance, also
putting the limits to the test.

In every folder there will be couple of resources
- the java file
- other ontologies (if exist)
- the output of the experiment
- Introduction
- COnclusion
- Environment
- pictures and charts
